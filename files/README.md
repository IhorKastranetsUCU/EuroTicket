# Flask застосунок з картою Польщі

## Опис
Веб-застосунок на Flask з інтерактивною картою Польщі, створеною за допомогою Folium. 
Користувачі можуть вводити станції відправлення та прибуття для пошуку маршрутів.

## Структура проекту
```
.
├── app.py                  # Основний файл Flask застосунку
├── requirements.txt        # Python залежності
├── templates/
│   └── index.html         # HTML шаблон
├── static/
│   ├── css/
│   │   └── style.css      # Стилі CSS
│   └── js/
│       └── main.js      # JavaScript для інтерактивності
```

## Встановлення

1. Встановіть необхідні бібліотеки:
```bash
pip install -r requirements.txt
```

2. Запустіть застосунок:
```bash
python app.py
```

3. Відкрийте браузер і перейдіть за адресою:
```
http://localhost:5000
```

## Функціональність

- ✅ Інтерактивна карта Польщі (Folium)
- ✅ Поле введення "Станція відправлення"
- ✅ Поле введення "Станція прибуття"
- ✅ Кнопка "Знайти маршрут"
- ✅ Відображення результатів пошуку
